# Tomer-s-Macropad
My very first hackpad through hackclub
Also my first hardware project ever! It was very fun, but I probably could have done things way better.

BOM:
6x Cherry MX Switches
2x SK6812 MINI Leds
1x XIAO RP2040
4x Through-hole 1N4148 Diodes
